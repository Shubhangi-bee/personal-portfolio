# 🌐 Personal Portfolio Website

A responsive personal portfolio website developed using **HTML, CSS, and JavaScript** to showcase my education, technical skills, projects, certifications, and contact information.

##  Live Demo

🔗 https://shubhangi-bee.github.io/personal-portfolio/

---

##  Features

- Responsive design for desktop and mobile devices
- Clean and modern user interface
- About Me section
- Semester-wise Academic Details
- Technical Skills section
- Projects showcase
- Resume download option
- Contact form with validation
- Smooth scrolling and interactive animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Shubhangi-bee/personal-portfolio.git
```

2. Open the project folder.

3. Double-click `index.html`

or

Open `index.html` using **Live Server** in VS Code.
